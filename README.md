# Valentine Microsite

Eine interaktive Ein-Seiten-Valentinsseite mit animiertem Herz-Intro, spielerischer Ja/Nein-Interaktion und schwebenden Herzen.  
Ideal, um jemandem auf kreative Weise eine kleine Überraschung zu machen oder eine persönliche Valentinsbotschaft zu teilen.

**Direkt nutzbar (GitHub Pages aktiv):**  
[https://alexyourguy.github.io/valentine/]

## Was macht diese Seite?

- Beim Öffnen erscheint ein pulsierendes Herz  
- Durch mehrfaches Klicken wächst das Herz und löst einen Übergang zur Frage-Seite aus  
- Auf der zweiten Ansicht erscheint die Valentinsfrage mit interaktiven Buttons  
- Bei „Ja“ verschwinden die Box und es steigen schwebende Herzen auf  
- Abschließend wird eine persönliche Affirmation angezeigt

Die Seite ist als **Single-Page** umgesetzt und funktioniert auf Desktop und Mobile.

## Features

- Interaktives Herz-Intro (Klick → Wachstum → Übergang)
- Flüssiger Übergang zur Frage-Ansicht
- „Nein“-Button weicht spielerisch aus
- „Ja“-Button triggert schwebende Herzen + Abschluss-Text
- Mobile-optimiert (inkl. Safari-Performance-Fixes)
- Light-Mode erzwungen (keine Farbverfälschung im Dark Mode)
- Keine externen Abhängigkeiten (reines HTML/CSS/JS)
