# Valentine Microsite

Interaktive Ein-Seiten-Valentinsseite mit Herz-Klick-Intro, Übergang und Animationen.

Live: [https://alexyourguy.github.io/valentine/]

## QR-Code
![Heart QR](assets/qr-heart.png)

## Features
- Interaktives Herz (Klick → Wachstum → Übergang)
- Ja/Nein-Box mit Animationen
- Floating Hearts
- Mobile-optimiert (inkl. Safari-Fixes)

## Nutzung
1. Repo klonen
2. `index.html` öffnen oder GitHub Pages aktivieren
